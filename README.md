# Aible Skills

박한민(Aible)의 AI 교육·강의·콘텐츠 제작용 Claude Code 플러그인 모음.

## 설치 (Claude Code CLI)

```bash
claude
> /plugin marketplace add aible-edu/aible-skills
> /plugin install vanilla-slide@aible-skills
> /plugin install edu-designer@aible-skills
```

## 플러그인 목록

| 이름 | 설명 |
|------|------|
| **vanilla-slide** | 순수 HTML/CSS/JS로 단일 파일 풀스크린 프레젠테이션 생성. 완성형 엔진 템플릿 동봉 — 전체화면(F)·목차(I)·화자 노트(N)·발표자 모드(S)·data-step 빌드 효과·인쇄(PDF 전 장) 대응. |
| **edu-designer** | 검증된 교육 모델(ADDIE·딕앤케리·메릴) 기반 커리큘럼 설계. 니즈 분석 인터뷰 → 모델 선택 → 모듈 구성 → 설계서 산출 워크플로우 — 블룸 텍사노미·교수법 레퍼런스 동봉. |
