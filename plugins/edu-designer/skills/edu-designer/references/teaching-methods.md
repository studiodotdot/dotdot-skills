# 교수방법 라이브러리

## 개요
커리큘럼 설계 시 각 모듈/차시에 적합한 교수방법을 선택하기 위한 참조 자료.
학습목표의 Bloom 수준, 학습자 수, 환경(온/오프라인)에 따라 최적의 방법을 선택한다.

## 교수방법 목록

### 1. 강의 (Lecture)
- **설명**: 교수자가 내용을 체계적으로 전달
- **적합**: 기억·이해 수준, 새로운 개념 도입, 대규모 인원
- **장점**: 시간 효율적, 많은 내용 전달 가능, 구조화 용이
- **단점**: 일방향, 집중력 저하 (20분 한계), 적용력 약함
- **팁**: 15-20분마다 활동 삽입, 핵심 질문으로 참여 유도

### 2. 토론 (Discussion)
- **설명**: 학습자 간 의견 교환을 통한 학습
- **적합**: 이해·분석·평가 수준, 다양한 관점 필요 시, 10-25명
- **장점**: 비판적 사고력 향상, 참여도 높음, 다양한 관점
- **단점**: 시간 소모, 일부 학습자 독점 가능, 준비 필요
- **형태**: 전체 토론, 소그룹 토론, 버즈 세션(2-3명), 패널 토론

### 3. 사례 연구 (Case Study)
- **설명**: 실제 또는 가상 사례를 분석하여 문제 해결 학습
- **적합**: 적용·분석·평가 수준, 의사결정 능력 개발
- **장점**: 현실감, 복합적 사고, 현업 연계
- **단점**: 좋은 사례 개발 어려움, 시간 소모
- **팁**: 자사 실제 사례 활용 시 효과 극대화

### 4. 역할극 / 롤플레이 (Role Play)
- **설명**: 학습자가 특정 역할을 맡아 상황을 시연
- **적합**: 적용 수준, 대인관계 기술, CS/영업/리더십 교육
- **장점**: 체험 학습, 감정 이해, 즉각적 피드백
- **단점**: 학습자 부담감, 비현실적 연기 가능성
- **팁**: 안전한 분위기 조성, 디브리핑 필수

### 5. 실습 / 워크숍 (Workshop / Hands-on)
- **설명**: 학습자가 직접 과제를 수행하며 학습
- **적합**: 적용·창조 수준, 기술/도구 습득, 산출물 필요 시
- **장점**: 학습 전이 높음, 동기 유발, 즉각적 성취감
- **단점**: 준비 시간, 개별 지도 필요, 속도 차이
- **팁**: 단계별 가이드 + 자율 시간 병행

### 6. 시뮬레이션 (Simulation)
- **설명**: 실제 업무 환경을 모사한 상황에서 학습
- **적합**: 적용·분석 수준, 위험/비용이 높은 실무 교육
- **장점**: 안전한 실패 가능, 현실감, 반복 연습
- **단점**: 개발 비용, 기술 의존
- **예시**: 콜센터 시뮬레이터, 가상 고객 응대, 모의 협상

### 7. 팀 프로젝트 (Team Project)
- **설명**: 소그룹이 공동으로 과제를 수행하여 산출물 도출
- **적합**: 분석·평가·창조 수준, 협업 능력 개발
- **장점**: 협업 경험, 복합적 역량, 현업 유사
- **단점**: 무임승차, 시간 소모, 평가 어려움
- **팁**: 역할 분담 명확히, 중간 점검, 개별+팀 평가 병행

### 8. 플립드 러닝 (Flipped Learning)
- **설명**: 사전 학습(영상/자료) + 교실에서 활동/토론
- **적합**: 이해+적용 수준 결합, 시간 제한 있을 때
- **장점**: 교실 시간 활용 극대화, 자기주도 학습
- **단점**: 사전 학습 미이행 리스크, 영상 제작 필요
- **팁**: 사전 학습 퀴즈로 이행률 관리

### 9. 게이미피케이션 (Gamification)
- **설명**: 게임 요소(포인트, 배지, 리더보드)를 교육에 적용
- **적합**: 기억·적용 수준, 반복 학습, 동기 유발 필요 시
- **장점**: 참여도·몰입도 향상, 경쟁 동기
- **단점**: 외적 동기 편향, 내용 경시 가능
- **팁**: 학습 목표와 게임 메커닉 정렬 필수

### 10. 멘토링 / 코칭 (Mentoring / Coaching)
- **설명**: 1:1 또는 소수 대상 개별 지도
- **적합**: 적용·통합 수준, 현업 전이 지원, 리더십 개발
- **장점**: 맞춤형, 깊은 학습, 관계 형성
- **단점**: 시간/인력 집약적, 확장 어려움

## 교수방법 선택 매트릭스

| Bloom 수준 | 소규모 (≤15명) | 중규모 (16-30명) | 대규모 (31명+) |
|-----------|--------------|----------------|--------------|
| 기억·이해 | 강의+퀴즈 | 강의+버즈세션 | 강의+플립드러닝 |
| 적용 | 실습, 롤플레이 | 워크숍, 사례연구 | 시뮬레이션, 게이미피케이션 |
| 분석·평가 | 사례연구, 토론 | 소그룹토론, 팀프로젝트 | 패널토론, 사례연구 |
| 창조 | 프로젝트, 멘토링 | 팀프로젝트, 워크숍 | 팀프로젝트, 경진대회 |
