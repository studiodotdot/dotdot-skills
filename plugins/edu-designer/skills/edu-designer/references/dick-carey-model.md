# 딕앤케리(Dick & Carey) 체제적 교수설계 모델

## 개요
Dick & Carey 모델은 ADDIE보다 더 세분화된 10단계 체제적 접근법이다.
교육 시스템의 각 구성요소 간 상호작용을 강조하며, 복잡한 교육 과정 설계에 적합하다.

## 10단계

### 1. 교수목표 설정 (Identify Instructional Goals)
- 요구 분석을 통해 학습자가 달성해야 할 최종 목표를 규명
- 조직의 성과 격차에서 출발
- 목표는 관찰·측정 가능한 행동으로 진술

### 2. 교수 분석 (Conduct Instructional Analysis)
- 목표 달성에 필요한 하위 기능(sub-skills) 분석
- 과제 유형 파악: 지적 기능, 인지 전략, 언어 정보, 운동 기능, 태도
- 학습 위계도(Learning Hierarchy) 작성 → 계열화의 근거

### 3. 학습자·맥락 분석 (Analyze Learners and Contexts)
- 출발점 행동(entry behaviors): 이미 알고 있는 것
- 학습자 특성: 동기, 태도, 선호 학습 양식
- 수행 맥락: 학습한 내용을 적용할 현업 환경
- 학습 맥락: 교육이 이루어지는 환경

### 4. 수행목표 진술 (Write Performance Objectives)
- ABCD 형식:
  - Audience (학습자)
  - Behavior (행동 — Bloom 동사)
  - Condition (조건)
  - Degree (기준)
- 예: "콜센터 상담원(A)은 불만 고객 시나리오가 주어졌을 때(C) 공감-해결-확인 3단계를 수행할 수 있다(B). 5개 중 4개 이상 정확히 수행한다(D)."

### 5. 평가 도구 개발 (Develop Assessment Instruments)
- 목표-평가 정렬(alignment) 확인
- 준거지향 평가(criterion-referenced) 설계
- 형성평가 + 총괄평가 도구 모두 개발

### 6. 교수 전략 개발 (Develop Instructional Strategy)
- 도입(동기유발, 목표 제시) → 전개(내용 제시, 연습, 피드백) → 정리(요약, 전이)
- 가네(Gagné)의 9가지 교수사태 활용 가능
- 매체·방법 선택

### 7. 교수 자료 개발 (Develop and Select Instructional Materials)
- 교수 전략에 따른 자료 개발/선정
- 교수자 가이드, 학습자 교재, 평가 도구

### 8. 형성평가 설계·실시 (Design and Conduct Formative Evaluation)
- 일대일 평가 → 소집단 평가 → 현장 시험
- 각 단계에서 수정·보완
- 설계 초기부터 평가를 내장하는 것이 핵심

### 9. 교수 수정 (Revise Instruction)
- 형성평가 데이터 기반으로 개선
- 약한 영역 식별 → 교수 전략/자료 수정
- 반복적 개선 프로세스

### 10. 총괄평가 설계·실시 (Design and Conduct Summative Evaluation)
- 교육 효과성 최종 판단
- 투자 대비 효과(ROI) 분석 포함 가능

## 적용 가이드

### 언제 딕앤케리 모델을 쓰나
- 다수의 이해관계자가 관여하는 대규모 교육 프로젝트
- 교육 시스템 전체를 설계해야 할 때 (단일 과정이 아닌 과정 체계)
- 하위 기능 분석이 중요한 복잡한 직무 교육
- 평가 기반 품질 보증이 필요한 인증/자격 교육

### 커리큘럼 설계 시 적용 포인트
- 2단계(교수 분석)의 학습 위계도 → 모듈 순서(계열화)의 핵심 근거
- 4단계(수행목표 ABCD) → 각 모듈의 학습목표 진술 형식
- 6단계(교수 전략) → 모듈별 교수방법 선택
- 딕앤케리 전체를 다 적용하기보다, 핵심 원리(위계 분석 + ABCD 목표 + 평가 정렬)를 채택
